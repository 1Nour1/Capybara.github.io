Capybara Cheat Sheet

by Nour Alaa Eldin Aly.

Navigation

  • visit('/sessions/new')
  • visit(root_path)

Clicking links and buttons

  • click_link('id, class or text')
  • click_button('id, class or text') → click_button('Sign in')
  • click('id, class or text') # Either a link or a button
  • page.find("#id or .class").click
  • link = "a[href='/contacts/1/edit']" then find(link).click

Interacting with forms

  • fill_in('id, class or text', with: 'text')

      • fill_in("contact_first_name", with: "Adam") → contact_first_name is an id

      fill_in('First Name', with: 'John')

      fill_in("hiring_contract_name", with: "NewHc", :visible => false) this is used in case the field isn't visible

      page.first("#contact_first_name").set("Alex") → this is used in case you have two fields with same id and you want to fill the first field

  • check('A checkbox')

      check('I accept the terms of the site')

  • uncheck('A checkbox')
  • choose('A radio button')

      • choose('Male')

  • select('Option', from: 'Select box')
  • find(:select, from, options).find(:option, value, options).select_option

      find('#contact_age').find(:xpath, 'option[2]').select_option

  • attach_file('Image', '/path/to/image.jpg')

Querying: Predicates / Matchers

  • expect(page).to have_button('Save')
  • expect(page).to have_no_button('Save') or expect(page).not_to have_button('Save')
  • expect(page).to have_link('users-index-nav')
  • expect(page).to have_content('Account created successfully')
  • expect(page).to have_selector()
  • expect(page).to have_table()
  • expect(page).to have_xpath()
  • expect(current_path).to eq(users_path)
  • page.should have_content('welcome')
  • page.should have_no_content('welcome') or page.should_not have_content('welcome')
  • page.has_css? / page.has_no_css?
  • page.has_xpath? / page.has_no_xpath?
  • page.has_content? / page.has_no_content?
  • page.has_link? / page.has_no_link?
  • page.has_button? / page.has_no_button?
  • page.has_field? / page.has_no_field?
  • page.has_select? / page.has_no_select?
  • page.has_checked_field? / page.has_unchecked_field?
  • expect(response).to have_http_status(200)

Scoping

  • within( '.class') do
      fill_in('First Name', with: 'John')
    end
  • within_fieldset
  • within_table

Response / Status Code

  • expect(response).to have_http_status(200)
  • page.status_code.should == 200

Debugging

  • save_and_open_page
